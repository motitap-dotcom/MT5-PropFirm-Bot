5. לעשות git pull ולקרוא את התוצאות

**כשאני שואלת "מה מצב הבוט?" או "תבדוק את השרת":**
→ תערוך את trigger-check.txt עם תאריך חדש → commit → push → חכה ל-workflow → pull → קרא תוצאות.
